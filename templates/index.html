Netflix search engine - Search by movie title






This project aims to build a movie recommendation mechanism and data analysis within Netflix.


In this project we plan to investigate trends relating to content on Netflix (TV shows and movies) available in different countries, identify similar content and
the type of content that is favored by Netflix and links between actors/directors.
The highlight of this project are the D3 VISUALIZATIONS as well as SEARCH ENGINE that uses machine learning, NLP (Natural Language Processing) - Text Classification using scikit-learn.

This is a text based recommendation engine. It indirectly finds the highest and smallest angle between two vectors by finding the largest cosine between the two vectors.
